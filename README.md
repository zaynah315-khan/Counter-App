# 🌻 Counter App – DOM Practice Project

A beginner-friendly **DOM (Document Object Model) Practice Project** built using **HTML, CSS, and Vanilla JavaScript**. This project focuses on learning how to manipulate web page elements dynamically through JavaScript while creating an interactive and visually appealing counter application.

---

## 🎯 Project Objective

The main goal of this project is to practice **DOM Manipulation** by:

- Selecting HTML elements using JavaScript
- Handling button click events
- Updating text dynamically
- Changing CSS styles through JavaScript
- Using conditional statements
- Working with arrays
- Creating an interactive user experience

---

## ✨ Features

- 🌻 Modern Glassmorphism-inspired UI
- ➕ Increment Counter
- 🎉 Custom Special Numbers
- 💬 Dynamic Message Box
- 🎨 Changes text color for special numbers
- ✨ Smooth counter animation
- 📱 Responsive design
- 💻 Beginner-friendly JavaScript

---

## ⭐ Custom Special Numbers

The application celebrates these custom special numbers:

**4 • 7 • 9 • 12 • 15 • 20 • 27 • 31 • 41 • 50**

Whenever the counter reaches one of these values, a congratulatory message is displayed automatically.
